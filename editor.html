🚀 How to Publish

StoryQuest uses GitHub Issues as its database. Follow these steps to make your story public:

  1. Open the StoryQuest GitHub repository: github.com/TheshanBMR/StoryQuest → Issues → New Issue
  2. Set the issue Title to:
  3. Copy the template below and paste it as the issue body:
  4. Add these Labels: published, your theme label, and any tag labels.
  5. Click Submit new issue. Note the issue number (e.g., #42).
  6. Your story will be live at: https://theshanbmr.github.io/StoryQuest/s.html?id=ISSUE_NUMBER
